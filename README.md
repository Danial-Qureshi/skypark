# Skypark

Basic prototype for just testing purpose
## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
yarn install
```

## Usage

```python

# for 'ios'
cd ios && pod install && yarn ios

# for 'android'
yarn android
```
